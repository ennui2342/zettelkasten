Strategic practice design concerns how to structure study sessions to maximise long-term retention rather than immediate performance. Two key variables are timing and ordering.

The spacing dimension: distributing practice across time rather than concentrating it in one session dramatically improves long-term retention. Ebbinghaus's forgetting curve shows why — each review session requires genuine retrieval, which re-encodes the memory with a shallower decay slope. Sessions spaced to arrive just before the memory would be lost produce the best return per review. Software implementations like Anki schedule reviews automatically using this principle.

The ordering dimension: mixing different topics or problem types within a session (interleaving) is superior to working through all instances of one type before moving to the next (blocked practice). In blocked practice, consecutive problems of the same type allow strategies to remain active in working memory, so each new problem doesn't require reconstructing the approach from scratch. Interleaving forces the learner to retrieve the relevant strategy for each problem anew, increasing difficulty during practice but producing substantially better performance on later tests.

Both spacing and interleaving are "desirable difficulties" — they make acquisition feel harder and slower in the moment, but this apparent obstacle is precisely what drives long-term retention. The common thread is that both require effortful processing rather than passive repetition.
