# Massed Practice

Massed practice — cramming — concentrates all study of a topic into a single session. It produces strong performance immediately after the session because the material stays active in working memory throughout, making each repetition easy. However, performance on tests given days or weeks later is substantially weaker than for students who distributed the same study time.

The problem is that massed repetitions bypass genuine retrieval. Because the material never actually fades between passes within the session, each review does not require pulling a decayed trace back from long-term memory. The retrieval practice mechanism — the primary driver of consolidation — is largely absent. The trace is repeatedly refreshed but never meaningfully re-encoded.

Students who cram consistently over-predict their delayed-test performance because the strong fluency they experience within the session feels like mastery. This metacognitive failure — confusing current accessibility with durable learning — is why the strategy persists despite its poor long-term results.
