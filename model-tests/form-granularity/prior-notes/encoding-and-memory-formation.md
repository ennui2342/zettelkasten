Memory formation depends critically on how incoming information is processed at encoding, not merely on how much exposure occurs. Levels of processing theory (Craik & Lockhart, 1972) showed that shallow processing — noticing surface features like font or rhyme — produces weaker memories than deep processing — attending to meaning, forming associations, elaborating on implications.

This explains why passive re-reading is a poor study strategy: it typically involves shallow processing of already-familiar material, generating a feeling of fluency without producing new encoding. The material feels known, but the memory trace is not meaningfully strengthened.

Encoding specificity is a related constraint: memory is easier to retrieve in conditions similar to those at encoding. Context, mood, and associated knowledge all form part of the encoded representation and function as retrieval cues. Information encoded with many associated cues is easier to retrieve across varied conditions.

New information is more durably encoded when it connects to existing knowledge: the prior structure provides hooks, makes the new information meaningful, and creates multiple retrieval pathways.

**Gap:** this note does not address what specific activities during encoding produce deep vs. shallow processing, or whether there is a single mechanism underlying different deep-processing strategies.
