## Genetic memory vs. emergent synthesis

Two notes contain content that clearly goes beyond either source paper: z20260322-002 and z20260322-003.

**z20260322-002** bridges procedural memory (from LEGOMem) with cognitive science frameworks about learning loops. The bridging language draws on established research on memory consolidation and the role of retrieval in encoding. Neither LEGOMem's paper nor EvoRoute would have generated this content from text alone — it requires training knowledge about learning science. This is genetic memory leaking into a SYNTHESISE note.

Is this a problem? **No, but it requires careful framing.** The synthesis delivers something a pure extraction system cannot: it places the paper's contribution in a richer conceptual frame that will help future papers about related topics retrieve it and connect it. This is analogous to an experienced researcher's annotation — the note captures not just what the paper says but what the reader (who has deep domain knowledge) understands it to imply. The benchmark-innovation.md framing of this as "curated long-term memory" is apt: the curation includes the model's own understanding, not just the paper's text.

The risk is that the benchmark's "temporal accuracy" task type will be confused by this — a task asking "what does the corpus say about X" may surface training knowledge rather than paper-specific claims. This is worth tracking but it is a task-design problem, not a pipeline problem.

**z20260322-003** (the trilemma note) is different. The specific insight — that the performance/cost/efficiency trilemma is the *reason* architecture and memory innovations must be co-designed — is emergent from having both papers available simultaneously. Neither paper makes this argument explicitly. This is the system doing what it should: identifying a relationship that is implicit in the cross-paper structure.

**Conclusion:** the pipeline is doing legitimate synthesis. The genetic memory contribution is present but appears to be additive (providing conceptual scaffolding) rather than replacing paper content. The distinction between "additive framing" and "contamination" should be checked explicitly at the inspection stage.
