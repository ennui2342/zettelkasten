# Spaced Repetition

Spaced repetition is the practice of distributing review sessions over time, timed to the forgetting curve. Memory strength decays exponentially after encoding; spacing reviews to arrive just before the memory would be lost produces maximum consolidation per session.

The standard explanation focuses purely on timing — you review the material again before it fully fades, which resets the curve with a shallower slope. But the timing is not the only mechanism. The gap itself matters because it determines how difficult the retrieval is: a memory retrieved with effort — after a longer interval, when the trace is weaker — produces a larger strengthening effect than an easy retrieval from a recently refreshed trace.

Spacing works in part because it makes retrieval hard. A longer gap means a weaker trace, which means the retrieval is more effortful, which means the re-encoding is more thorough. Two reviews separated by a week each produce a stronger memory than four reviews separated by a day each, even if total review time is identical. The difficulty is not a side-effect of spacing — it is a mechanism. Retrieval difficulty is itself a consolidating force.

This resolves the open question of whether spacing interval alone explains the benefit: it does not. The interval produces difficulty, and the difficulty produces consolidation. Both are necessary for the full effect. Systems that reduce retrieval difficulty (e.g. by providing too many hints or showing the answer too readily) will underperform even with optimal timing.
