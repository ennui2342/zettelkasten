Human memory is not a single system but a collection of distinct stores with different properties, capacities, and timescales.

Working memory (sometimes called short-term memory) is the temporary, limited-capacity system that holds information currently in use. Baddeley and Hitch's model identifies a central executive, a phonological loop for verbal information, a visuospatial sketchpad for visual and spatial information, and an episodic buffer that integrates across systems. Capacity is famously limited — Miller's estimate of 7 ± 2 items has been revised downward to roughly 4 chunks in more recent work. Information in working memory decays within seconds without active rehearsal.

Long-term memory is the system for durable storage. It has effectively unlimited capacity and retains information from minutes to decades. Encoding from working memory to long-term memory is not automatic — it requires processing depth, repetition, or emotional salience. Long-term memory is further divided into declarative memory (facts and events that can be consciously reported) and non-declarative memory (skills, habits, and conditioned responses that operate implicitly). Declarative memory splits further into semantic memory (context-free facts) and episodic memory (events situated in time and place).

Transfer between the systems flows in both directions: working memory draws on long-term memory for meaning and context; long-term memory is built up through working memory processing. The constraint on the system is often the bottleneck of working memory capacity, which limits how much new material can be actively processed and thus encoded at once.
