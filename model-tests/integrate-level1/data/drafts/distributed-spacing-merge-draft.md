# Distributed Practice and Spacing

The educational psychology literature uses the term "distributed practice" for the strategy of spreading study sessions across a curriculum rather than massing them. The cognitive psychology literature uses "spacing effect" or "spaced repetition" for the same principle studied in controlled laboratory conditions. These are not different phenomena — they are the same empirical finding described using the terminology of two different disciplines.

Both describe the robust finding that the same total amount of practice produces dramatically better long-term retention when distributed over time than when concentrated in a single session. Both explain this by reference to the forgetting curve: distributed sessions arrive when the trace has decayed, making retrieval effortful, which is precisely what drives re-encoding. Both cite Ebbinghaus as the originating study and Cepeda et al.'s meta-analysis as the best systematic evidence.

The terminological difference reflects disciplinary lineage. Cognitive psychologists studying memory processes in the lab call it the spacing effect. Educational psychologists studying instructional design in classrooms call it distributed practice. Practitioners building software for self-study call it spaced repetition. The implementations differ — the spacing effect is typically studied with fixed intervals, distributed practice with curriculum sequencing, spaced repetition with adaptive scheduling — but the underlying principle is identical.

Treating these as separate topics in a knowledge base would be redundant. Any note on one is effectively a note on the other, just with different citation conventions and practical framings.
