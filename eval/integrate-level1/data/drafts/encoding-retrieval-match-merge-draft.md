# Encoding-Retrieval Match

Memory retrieval succeeds most reliably when the conditions at retrieval match the conditions at encoding. This encoding-retrieval match principle — encoding specificity (Tulving) — unifies what appear to be separate phenomena in the literature under a single account.

Context-dependent memory (the finding that recall is better when tested in the same environment as learning) is an instance of encoding-retrieval match: the physical and cognitive context present at encoding is encoded as part of the memory trace, and that context functions as a retrieval cue. Retrieving in a matching context provides those cues; retrieving in a different context means those cues are absent.

Retrieval cue effectiveness is also encoding-retrieval match: a cue is effective to the degree that it was encoded with the target. A cue that was present in the cognitive environment at encoding has high informational overlap with the trace; a cue introduced only at retrieval has low overlap and is less effective. The reason elaborative encoding creates robust memories is that it creates many potential retrieval cues — every association formed at encoding is a potential route back to the target.

These two literatures — context-dependent memory (which focuses on what happens at encoding) and retrieval cues (which focuses on what happens at retrieval) — are describing the same underlying principle from opposite ends. The memory trace encodes whatever was present at encoding; retrieval succeeds when it reinstates those encoded elements. The symmetry is exact: good encoding creates good cues; good cues access well-encoded traces.
