External memory systems — notebooks, databases, search engines, digital note tools — extend human cognitive capacity by off-loading storage and retrieval to an external substrate. The extended mind thesis (Clark & Chalmers, 1998) argues that when an external tool is reliably available, integrated into workflow, and trusted, it functions as a genuine component of the cognitive system rather than merely an aid to it.

The Google Effect (Sparrow et al., 2011) showed that people encode less information when they expect it to be retrievable from an external source. This can be adaptive — off-loading storage frees attention for processing — but the mechanism implies a trade-off: information that is only stored externally is not encoded internally, and may not be available when the external source is absent.

The design question for knowledge tools is when off-loading is productive (freeing resources for higher-order work) versus counterproductive (substituting for the internal encoding that deeper understanding requires).

**Open question:** what properties of an external system determine whether interaction with it produces genuine encoding, as opposed to mere storage in an inaccessible external form?
