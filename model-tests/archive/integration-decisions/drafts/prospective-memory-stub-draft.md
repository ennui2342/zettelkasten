# Prospective Memory

Prospective memory is the ability to remember to perform an intended action at some point in the future — "remember to call the dentist tomorrow," "take the medication with dinner," "send the report by Friday." It is distinct from retrospective memory (remembering past events or learned facts) because the goal is not to recall what happened, but to initiate a future behaviour at the right time.

Prospective memory failures are among the most common everyday memory errors. They are responsible for missed appointments, forgotten medications, and overlooked commitments. Crucially, the task is not forgotten between the intention and the execution — it simply fails to be triggered at the right moment.

Event-based prospective memory (triggered by an external cue, like "when I see the dentist sign, remember to call") is generally more reliable than time-based prospective memory (triggered by an internal clock check, like "remember at 3pm"). The presence of a salient environmental cue reduces the demand on internally-generated monitoring.

Implementation intentions — specifying in advance the exact when, where, and how of the intended action ("when I sit down for dinner, I will take the medication") — substantially improve prospective memory success by creating an if-then link between the cue and the action.
