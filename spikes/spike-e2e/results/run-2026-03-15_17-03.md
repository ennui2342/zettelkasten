### 1. Active Recall and Retrieval Practice

**Draft note:**

Active recall and retrieval practice are two names for the same concept, originating from different communities. "Active recall" is the term favoured by self-directed learners, flashcard communities, productivity writers, and popular learning guides, having been developed empirically by practitioners often without reference to academic work. "Retrieval practice" is the term used in experimental psychology, education research, and the desirable difficulties literature, studied under controlled conditions. Both describe the same behaviour: attempting to retrieve target information from memory without consulting the source material. Both explain it through the same mechanism: effortful retrieval strengthens the memory trace by forcing reconstruction. The shared core finding is that generating information from memory produces stronger retention than re-studying the same information, and the shared practical implication is to replace re-reading and passive review with active generation and self-testing. The evidence base is unified — every experimental demonstration of the testing effect simultaneously supports active recall, and every practical implementation of active recall is an instance of retrieval practice.

Active recall is a study technique in which the learner attempts to retrieve target information from memory without consulting the source material. Rather than re-reading a textbook or reviewing highlighted notes — activities that feel productive but rely on recognition rather than genuine retrieval — active recall requires the learner to produce an answer from an incomplete or degraded cue.

Common implementations include flashcards (where the cue is presented and the learner must produce the answer before flipping), practice questions, blank-page recall (closing all materials and writing everything remembered about a topic), and the Feynman technique (explaining a concept from memory to identify gaps). The defining feature is that the learner must generate the target rather than recognise it.

The benefit of active recall over passive review is well-established: generating information from memory strengthens the memory trace more than re-exposing oneself to the information. The process of effortful retrieval lays down additional associative pathways and deepens integration of the material with prior knowledge. Failure during retrieval — where the learner cannot produce the answer — is itself informative, identifying which material requires further study. Active recall is widely recommended in self-directed learning communities and productivity literature as the single most effective individual study strategy.
